What it does

An autonomous library
that builds itself

Feed it a reading list. Walk away. Come back to a hard drive full of organised, catalogued, deduplicated books — sourced from Project Gutenberg, Internet Archive, and Open Library simultaneously.

The canon queue drip-feeds 15 works per hour into a SQLite want-list. The agent picks them up, finds them across three APIs, validates the download, runs local AI analysis via Ollama, and files them away by era and subject.

Capabilities
  • 01Multi-source parallel search across Gutenberg, Archive.org & Open Library
  • 02SHA-256 deduplication with 85% fuzzy match threshold
  • 03Local Ollama (mistral:7b) metadata analysis & categorisation
  • 04Organised file system — great-books/{era}/{title}.epub
  • 05Four n8n automation workflows + systemd 24/7 service
  • 06Discord progress embeds & failure alerts
  • 07Exponential backoff retry — 5 s → 15 s → 60 s
